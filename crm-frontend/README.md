# PulseCRM Frontend

React + Vite + Tailwind UI for the Xeno take-home assignment.

## Setup

```bash
npm install
npm run dev
```

Opens **http://localhost:5173** (proxies `/api` to backend on port 5000).

## Production

```bash
# Set VITE_API_URL to your deployed backend
npm run build
npm run preview
```

Deploy `dist/` to Vercel.
