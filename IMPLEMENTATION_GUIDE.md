```

    ↓

    ├─ Discount: Adjust percentage
```

## 🔧 How to Use (Step-by-Step)

### Step 1: AI Copilot - Diagnosis
1. Go to **AI Copilot** tab
2. Enter your campaign goal (e.g., "Win back at-risk shoppers with 15% off")
3. Click **Diagnose** button
4. AI analyzes the goal and identifies target segment

### Step 2: Message Generation
1. Click **Generate Message & Channel Strategy**
2. AI generates:
   - Personalized message
   - Optimal channel (based on segment)
   - Recommended discount
   - Alternative variants

### Step 3: Full Customization
The **Campaign Customizer** now appears with:

#### A. Edit Message
- Click **Edit** button
- Modify the message text
- Keep `{name}` placeholder for personalization
- Click **Lock** when done

#### B. Change Channel
- Select from 4 options
- Each channel optimized for segment:
  - **WhatsApp**: Highest engagement, personal
  - **SMS**: Fast delivery, action-oriented
  - **Email**: Rich formatting, detailed
  - **RCS**: Rich media, interactive

#### C. Adjust Discount
- Use slider or type percentage
- Range: 0-100%
- See real-time preview update

#### D. Select Customers (Optional)
- Click **Show Segment** button
- See all customers in segment
- View their: name, email, spend, order count, vitals
- Check customers to select manually
- Leave unchecked to use entire segment

### Step 4: Launch
- Click **Launch** button
- Campaign sent to:
  - All segment customers (if none selected)
  - OR selected customers only (if you chose specific ones)
- Messages appear in Live Stream
- Real-time analytics update

## 📈 Analytics Tracked

For each campaign:
- **Target Count**: Total customers receiving campaign
- **Delivered Count**: Successfully sent messages
- **Open Count**: How many opened message
- **Click Count**: How many clicked link
- **Conversion Count**: Actual purchases
- **Open Rate**: %= Opened / Targeted
- **Click Rate**: %= Clicked / Targeted
- **Revenue Attributed**: Sales from this campaign

## 🆘 Troubleshooting

**Issue**: "Gemini quota exhausted"
→ Set `USE_MOCK_GEMINI=true` in .env

**Issue**: "No customers in segment"
→ Run seed.js to populate test data: `npm run seed`

**Issue**: "Channel selector not showing"
→ Clear browser cache, reload frontend

**Issue**: "Messages not appearing in live stream"
→ Check channel simulator is running on port 5001

**Issue**: "Discount not showing in message"
→ Discount is tracked in campaign metadata, not in message text
